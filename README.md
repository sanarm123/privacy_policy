Data may be stored locally on the device and in cloud services (e.g., Firebase services) configured by your app deployment.

Cloud data is processed according to your configured backend provider policies.

We do not sell personal data.

With service providers needed to operate the app (e.g., cloud hosting, authentication, storage)

The app requests Bluetooth permissions to:

Discover paired printers/scanners
Connect and print receipts/reports
Save selected device identifiers for faster reconnect
